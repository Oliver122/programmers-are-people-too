## How It Works ğŸ› ï¸

The extension monitors VS Code's diagnostic system (the same system that shows red/yellow underlines) and celebrates when things improve:

1. **Tracks Changes**: Monitors diagnostic events across all open files
2. **Detects Improvements**: Compares current vs. previous error/warning counts  
3. **Celebrates Progress**: Shows positive feedback when issues are resolved
4. **Logs Everything**: Detailed tracking in output panel and console
